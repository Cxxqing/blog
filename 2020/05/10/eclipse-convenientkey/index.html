eclipse快捷键

发表于 2020-05-10 更新于 2020-06-15
本文字数： 202 阅读时长 ≈ 1 分钟

常用的快捷键：

添加单行注释： Ctrl + / 取消：Ctrl + /
添加多行注释：Ctrl + Shift + / 取消：Ctrl + Shift +
向下复制一行：Ctrl + Alt + 向下
向上复制一行：Ctrl + Alt + 向上
向上移动代码：Alt + 向上
向下移动代码：Alt + 向下
删除当前行：Ctrl + D
格式化：Ctrl + Shift + F
智能提示：Alt + /
更改变量名称：Alt + Shift + R
弹出源代码的相关选项： Alt+Shift+S
